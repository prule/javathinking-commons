Validation

'Validation' is a small package to aid in validation. It aims to be simple and effective, but its simplicity may not be appropriate for all projects.

Basic concepts

  • A Validator has one or more ValidationRule
  • A Validator may
    • 'fail fast' - stop after the first reported error
    • or 'fail slow' - process all rules regardless of any reported errors
  • A ValidationRule takes an input and returns a list of errors detected
